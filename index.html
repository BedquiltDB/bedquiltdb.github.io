BedquiltDB

A Json store on PostgreSQL

What?

BedquiltDB is a PostgreSQL extension and set of code libraries, which wraps the PostgreSQL jsonb data type in a slick programatic API inspired by NoSQL data stores.

Why?

Many developers prefer the code-centric APIs of NoSQL stores like MongoDB, especially in the early stages of a project, but would like to still have all the power of an SQL database available without going through a painful migration.

BedquiltDB aims to provide developers with a powerful tool, both in early stages of a project, and when explicitly dealing with loosely-structured data, while providing an easy migration path into SQL-land once the schemas fall into place.
