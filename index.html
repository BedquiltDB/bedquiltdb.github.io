BedquiltDB

A Json store on PostgreSQL

CREATE EXTENSION bedquilt;

What?

BedquiltDB is a PostgreSQL extension and set of code libraries, which wraps the PostgreSQL jsonb data type in a slick programatic API inspired by NoSQL document stores.

Why?

Many developers prefer the code-centric APIs of NoSQL stores like MongoDB, especially in the early stages of a project, but would like to still have all the power of an SQL database available without going through a painful migration.

BedquiltDB aims to provide developers with a powerful tool, both in early stages of a project, and when explicitly dealing with loosely-structured data, while providing an easy migration path into SQL-land once the schemas fall into place.

Examples

Python

bq = pybedquilt.BedquiltClient()
projects = bq['test']['projects']

projects.insert({
    '_id':   "BedquiltDB",
    'description': "A ghastly hack.",
    'quality': "pre-alpha",
    'tags': ["json", "postgres", "api"]
})

early_projects = projects.find(
    {'quality': 'pre-alpha'})

api_projects = projects.find(
    {'tags': ['api']})

cool_project = projects.find_one_by_id('BedquiltDB')

Clojure

(require '[bedquilt.core :as bq])

(def db
  (bq/get-db
    {:db-host "localhost"
    :db-name "bedquilt_test"
    :user "some_user"
    :password "password"}))

(bq/save! db :people
  {:name "John Doe", :age 54})

(bq/find-one-by-id db :people
  "sarah.jones@example.com")

Features

Core features:

  • Easy-to-install PostgreSQL extension
  • Automatically create collections on first write
  • Insert and Save data to a collection
  • Automatic generation of id fields for inserted documents
  • Find data documents from collections
  • Remove data from collections


Client (Driver) Libraries:

  • Python (alpha)

Roadmap

Core features:

  • Add constraints to collections
  • Add indexes to collection fields
  • Advanced query operations
  • Update, with partial modification of documents
  • Collection namespaces


Client (Driver) Libraries:

  • Python
  • Clojure
  • Javascript
  • Ruby
  • Scala
  • Go
  • And many more...

Projects

  • bedquilt-core - The main PostgreSQL extension.
  • pybedquilt - The official Python driver for BedquiltDB.
